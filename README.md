# Fubell

An unofficial Typst template for National Taiwan University (NTU) theses and dissertations.

## Description

Fubell is a Typst template designed to help NTU students format their theses and dissertations according to university guidelines. Powered by Typst, this template provides an easy-to-use alternative to traditional LaTeX templates while maintaining professional formatting standards.

The template name is inspired by the iconic Fu Bell (傅鐘) on NTU campus.

### Features

- **NTU-compliant formatting**: Adheres to National Taiwan University thesis formatting requirements
- **Bilingual support**: Support for both English and Traditional Chinese content
- **Modern typesetting**: Leverages Typst's powerful and intuitive syntax
- **Easy customization**: Simple configuration for title pages, abstracts, and acknowledgments
- **Automatic formatting**: Pre-configured spacing, margins, and typography
